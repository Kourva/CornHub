1. **Clone repository**
```bash
git clone https://github.com/Kourva/CornHub
```
2. **Install requirements**
```
pip install flet
```
3. **Run**
```
flet run main.py
```
> [!TIP]
> If you want to package the app...
```bash
flet pack main.py
```
